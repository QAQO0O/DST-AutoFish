# 饥荒联机版-自动钓鱼模组

## 功能

1. 普通淡水鱼竿：
   1. 自动钓鱼
   2. 自动装备鱼竿
   3. 自动做鱼竿（身上有材料）
2. 玻璃钓竿（能力勋章的内容）：
   1. 自动钓鱼
   2. 自动装备鱼竿（优先级比淡水鱼竿高）
   3. 身上带有蜘蛛丝时自动补充耐久
   4. 自动根据当前钓鱼的池塘填充鱼饵
3. 如果鱼塘里面没鱼了，则自动切换附近的鱼塘，如果附近没其他鱼塘则重新钓鱼，检测时间间隔为游戏内的半天，即半天没上钩则重新钓鱼
